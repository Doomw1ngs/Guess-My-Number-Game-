A simple number guessing game built with vanilla JavaScript, HTML, and CSS.

The game generates a random number between 1 and 20, and the player must guess it. Feedback is provided after each guess to help the player find the correct number.

How It Works

A random secret number is generated when the game starts

The player enters a number between 1 and 20

The game responds with:

Too high

Too low

Correct number

The score decreases with each wrong guess

Clicking Again resets the game and generates a new number

Features

Random number generation

Score tracking

Game reset without page reload

Input auto-clear for better user experience

Simple and clean UI

HTML

CSS

JavaScript (Vanilla)
